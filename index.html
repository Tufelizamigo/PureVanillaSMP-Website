How to join

Java Java Edition

Any recent Java client works. If you're on a cracked launcher, that's fine here.

  1. Open Minecraft → Multiplayer
  2. Click Add Server
  3. Paste purevanillasmp.xyz as the address
  4. Join and say hi in chat

Bedrock Bedrock Edition

Bedrock players come through Geyser. You'll need the address and port.

  1. Servers tab → Add Server
  2. Address: purevanillasmp.xyz
  3. Port: 14061
  4. Save & join

Tip: the .xyz address works whenever the server's awake. If it won't connect, it may be starting up — give it a minute and retry.

Don't kill new players, players collecting their kits, or people mid-farm.

What this is

Pure Vanilla is a small, long-running survival world that doesn't get wiped. The point is the opposite of most servers — no economy plugins, no ranks you buy, no minigame hub. Just the base game, a shared map, and whatever your group decides to build. Java and Bedrock players share the same world through Geyser, so nobody gets left out.
